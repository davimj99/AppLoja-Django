## 📌 Sobre o projeto

Este é um sistema de **e-commerce completo desenvolvido em Django**, com arquitetura modular, organizado em apps separados como:

- produtos
- carrinho
- pedidos
- clientes
- pagamentos
- estoque
- backend (configurações gerais)

O projeto simula uma loja online com funcionalidades reais de compra, gerenciamento de produtos e controle de pedidos.

- 🛍️ Listagem de produtos

## 🧱 Arquitetura

O projeto segue uma arquitetura modular baseada no framework Django, organizada em múltiplos apps independentes.

Cada app é responsável por uma funcionalidade específica do sistema, garantindo melhor separação de responsabilidades, escalabilidade e facilidade de manutenção.

### 📂 Estrutura principal:

- **produtos** → gerenciamento de produtos e categorias  
- **carrinho** → controle de itens adicionados pelo usuário  
- **pedidos** → criação e acompanhamento de pedidos  
- **clientes** → cadastro e gestão de usuários/clientes  
- **pagamentos** → processamento e controle de pagamentos  
- **estoque** → controle de disponibilidade de produtos  
- **backend** → configurações gerais do projeto (settings, urls, wsgi)

Essa organização permite que o sistema cresça de forma estruturada, facilitando a adição de novas funcionalidades sem impactar o restante da aplicação.
